Dictation Example

This is an example that uses a Dictation object to get text input into a Rocky app.

What's going on?

This example uses the Web Speech API to obtain dictated text from the user.

A Dictation object is created with new Dictation();.

Next, the start() function is called and provided with two callbacks - successCallback that provides the transcribed text, and failureCallback that provides a string reason for any errors that occur.
